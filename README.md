# Automated Data Cleaning & EDA Tool

This Streamlit application automates data cleaning and exploratory data analysis for CSV files.

## Features
- Upload CSV files
- Missing value handling (mean, median, drop rows)
- Outlier detection using IQR (numeric columns only)
- Data preview before and after cleaning
- Download cleaned CSV

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

## Deployment
Deployed on Streamlit Community Cloud.
